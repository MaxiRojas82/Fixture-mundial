Política de Privacidad

MaxFixture – Mundial 2026
Última actualización: 8 de junio de 2026

1. Información que recopilamos

MaxFixture no recopila ni almacena datos personales de sus usuarios. La aplicación no requiere registro, inicio de sesión ni cuenta de ningún tipo.

2. Uso de internet

La app se conecta a los siguientes servicios externos para mostrar información del torneo:

  • football-data.org – para obtener fixtures, resultados y posiciones del Mundial FIFA 2026.
  • Google Gemini API – para generar análisis breves de los partidos (texto generado por IA).
  • flagcdn.com – para cargar imágenes de banderas de los equipos.

Estas conexiones solo se realizan para mostrar contenido deportivo en pantalla. No se envía ningún dato del usuario a estos servicios.

3. Notificaciones push

Si el usuario activa las notificaciones, la app puede recibir alertas de goles y eventos del partido a través de Firebase Cloud Messaging (FCM) de Google. FCM no recopila datos personales identificables.

4. Permisos de Android

  • INTERNET – necesario para obtener datos del torneo.
  • RECEIVE_BOOT_COMPLETED – para reiniciar el servicio de notificaciones al encender el dispositivo.
  • VIBRATE – para vibrar al recibir una notificación.

5. Servicios de terceros

Las políticas de privacidad de los servicios utilizados están disponibles en:

  • football-data.org
  • Google (Gemini / Firebase)

6. Contacto

Si tenés alguna pregunta sobre esta política, podés escribirnos a: mrojas@handing.co

7. Cambios a esta política

Cualquier cambio será publicado en esta página con la fecha de actualización.
